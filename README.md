# dada-devs-cohort-5

Welcome to the repo for **Dada Devs Cohort 5**! This is our shared space for the cohort — collaborating, learning, and tracking who's part of the group.

## Getting Started

1. Fork this repository.
2. Clone your fork locally.
3. Create a new branch for your changes.

## Contributing

Before making any other contributions, every member is required to add themselves to [Contributors.md](Contributors.md).

To add yourself:

1. Open [Contributors.md](Contributors.md).
2. Add a new entry following the existing format:

   ```
   [your name] @yourhandle

   "why I'm here ..."
   ```

3. Commit your change, push to your fork, and open a pull request against `main`.

Please keep your entry concise — a short line for your name/handle, and one sentence on why you're here.

## Code of Conduct

Be respectful, be helpful, and support your fellow cohort members.
